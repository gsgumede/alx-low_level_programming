# 0x0A. C - argc, argv
## General
- How to use arguments passed to your program
- What are two prototypes of `main` that you know of, and in which case do you use one or the other  
- How to use `__attribute__((unused))` or `(void)` to compile 
## Requirements
- Ubuntu 20.04 LTS
- gcc gnu89
